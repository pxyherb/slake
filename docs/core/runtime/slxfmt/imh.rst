Image Header
============

+--------------+-------+------------------------------------------------------------+
| Offset (dec) | Type  | Description                                                |
+--------------+-------+------------------------------------------------------------+
| 0-3          | u8[4] | Magic number, always is 0x58414c53 ('SLAX') in big endian. |
+--------------+-------+------------------------------------------------------------+
| 4            | u8    | Module flags.                                              |
+--------------+-------+------------------------------------------------------------+
| 5            | u8    | Format version.                                            |
+--------------+-------+------------------------------------------------------------+
| 6-7          | u8[2] | Reserved.                                                  |
+--------------+-------+------------------------------------------------------------+
